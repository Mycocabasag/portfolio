Logo Blur Implementation Guide

Option 1: CSS Only (Recommended)

Add this CSS to your existing stylesheet or create a new one:


/* Blur the Medical Center logo */
.logo, .brand, .medical-center-logo {
    filter: blur(8px) !important;
    -webkit-filter: blur(8px) !important;
}

/* Target images in left container */
.left-container img, .left-panel img {
    filter: blur(8px) !important;
    -webkit-filter: blur(8px) !important;
}

Option 2: JavaScript Implementation

Include the blur-logo.js file in your HTML:


<script src="blur-logo.js"></script>

Option 3: Inline Style (Quick Fix)

Add this directly to your HTML head section:


<style>
.logo, .brand, [class*="logo"] {
    filter: blur(8px) !important;
    -webkit-filter: blur(8px) !important;
}
</style>

Option 4: Browser Developer Tools (Temporary)

For immediate testing, open browser developer tools (F12) and add this to the console:


// Blur all images in the left half of the screen
document.querySelectorAll('img').forEach(img => {
    const rect = img.getBoundingClientRect();
    if (rect.left < window.innerWidth / 2) {
        img.style.filter = 'blur(8px)';
        img.style.webkitFilter = 'blur(8px)';
    }
});
